Project Overview

This project demonstrates an innovative autonomous drone-to-drone docking system utilizing infrared (IR) LED technology for precise positional control during flight. The primary objective is to enable a drone to autonomously dock onto another drone mid-air, with exceptional accuracy and reliability.

At the core of the system, infrared LEDs operating at a 38kHz carrier frequency provide distinctive identification patterns for orientation and positioning. The target drone is equipped with multiple IR LEDs, each transmitting unique patterns. The landing drone employs IR sensors to detect and interpret these patterns, using an ESP32 microcontroller that processes signals with advanced filtering techniques like Kalman filters.

Communication between drones and their Ardupilot flight controllers is achieved through MAVLink and ESP-NOW protocols, enabling real-time data exchange and drone-to-drone interactions. A custom-developed adaptive Proportional-Derivative (PD) controller, implemented in Lua scripting language within the Ardupilot ecosystem, dynamically adjusts the docking maneuvers based on altitude differences and real-time position errors, ensuring smooth and precise docking operations.

This autonomous docking system lays a solid foundation for advanced UAV applications, including autonomous drone swarms, aerial refueling, precision agriculture, and automated inspection tasks.
